# 机器学习面试笔记

Accuracy 表示预测正确的比例，但在类别不平衡时容易被多数类主导。例如正类非常少时，模型全部预测为负类也可能有很高 accuracy，但这并不代表模型能识别正类。

Precision 表示预测为正的样本中有多少是真的正。Recall 表示真实为正的样本中有多少被模型找出来。F1 是 precision 和 recall 的调和平均，适合同时关注查准和查全的场景。

ROC-AUC 衡量不同分类阈值下 TPR 和 FPR 的权衡。PR-AUC 衡量 precision 和 recall 的权衡，在正类稀少的场景通常更有参考价值。

随机森林是 bagging 思路，多棵树并行训练，主要降低方差。GBDT 是 boosting 思路，多棵树串行训练，后一棵树拟合前面模型的残差或负梯度，主要降低偏差。

Loss 是训练时优化的目标，通常要求可导。Metric 是最终评估或业务关注的指标，不一定可导。训练时可以优化交叉熵，但评估时关注 F1、AUC 或业务收益。
